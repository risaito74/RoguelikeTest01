# リスタート機能 実装検証手順

## 変更内容
- **GameManager.cs**:
    - `Restart Button` フィールドを追加。
    - ゲーム開始時に非表示、プレイヤー死亡時に表示する制御を追加。
    - ボタン押下時に現在のシーンをリロードする処理を追加。

## 検証手順

### 1. Unityエディタ設定
> [!IMPORTANT]
> **ボタンの作成と割り当てが必要です**
> 1. Hierarchyで右クリック -> UI -> Button (Legacy または TextMeshPro) を作成し、名前を `RestartButton` に変更してください。
>    - ボタンのテキストは "Restart" などに設定し、画面の適当な位置（中央など）に配置してください。
> 2. `GameManager` オブジェクトの Inspector を開き、`Restart Button` フィールドに作成した `RestartButton` をドラッグ＆ドロップしてください。

### 2. プレイ検証
Unityエディタで再生ボタンを押し、以下の動作を確認してください。

1.  **初期状態確認**
    - ゲーム開始時、リスタートボタンが表示されていない（非アクティブになっている）ことを確認。
2.  **死亡時表示確認**
    - スライムに体当たりして死亡（GameOver）する。
    - ゲームオーバー表示とともに、リスタートボタンが表示されることを確認。
    - **右上のHP表示の文字色が赤色に変わることを確認。**
3.  **リロード確認**
    - リスタートボタンを押す。
    - ゲームが最初から始まり（HP全快、初期位置）、リスタートボタンが消えることを確認。
    - **HP表示の文字色が白に戻ることを確認。**
